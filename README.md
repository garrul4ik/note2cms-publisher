# note2cms Publisher

Плагин для публикации заметок Obsidian в note2cms.

## Установка
1. `npm install`
2. `npm run build`
3. Скопируйте `main.js`, `manifest.json`, `styles.css` в `.obsidian/plugins/note2cms-publisher/`

## Функции
- Офлайн очередь
- Публикация только по WiFi
- Подтверждение на мобильных
- Массовая публикация
